Phorm_Field_ImageUpload

A Phorm_Field_FileUpload that is pre-configured for images. Valid types are png, gif, jpeg/jpg. Returns a Phorm_Type_Image instance instead of a Phorm_Type_File instance. Identical to the Phorm_Field_FileUpload in all other ways.
